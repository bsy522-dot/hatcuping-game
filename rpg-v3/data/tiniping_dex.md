# 티니핑 도감 (Tiniping Pokedex) - 41종

## 개요
`tiniping_dex.js`는 `캐치!티니핑` 시즌 1~5 및 극장판 `사랑의 하츄핑` 원작 캐논을 기반으로 한 포켓몬식 RPG용 데이터이다. 전역 `window.TINIPING_DEX` 배열로 노출되며 41개 엔트리를 담는다.

## 엔트리 필드

| 필드 | 타입 | 설명 |
|------|------|------|
| no | int | 도감 번호 1~41 |
| name | string | 티니핑 이름 (한글) |
| class | enum | `royal` / `normal` / `villain` / `legend` |
| type | string[] | 감정 타입 1~2개 |
| emotion | string | 대표 감정 1개 |
| season | string | 원작 등장 시즌 (S1~S5, 극장판) |
| baseStats | object | hp / atk / def / spAtk / spDef / spd |
| skills | string[] | 4~5개의 기술명 |
| evolveLv | int/null | 진화 레벨 (없으면 null) |
| evolveTo | string/null | 진화 대상명 |
| habitat | string[] | 서식지 맵 ID 1~3개 |
| rarity | 1~5 | 등급 |
| desc | string | 두 문장 캐논 기반 설명 |

## 4분류 (Class)

### 로열 (Royal) - 12종
왕족 혈통의 강력한 티니핑. 로미 공주·제니 등 주인공 파트너 계열. **스탯 총합 평균 ≈ 430~460**, 희귀도 5. 각자 고유 감정을 상징한다.

### 일반 (Normal) - 20종
마음마을·각지에 흩어진 평범한 티니핑. 감정 유발형. **스탯 총합 평균 ≈ 280~340**, 희귀도 2~3.

### 빌런 (Villain) - 6종
악역 티니핑. 어둠 계열이 주류이며 로열을 노린다. **스탯 총합 평균 ≈ 380~445**, 희귀도 3~4.

### 레전드 (Legend) - 3종
전설의 티니핑. 한 세대에 한 번만 나타나는 존재들로 스탯 총합 **600+** 확정. 희귀도 5.

| class | 평균 스탯 총합 | 희귀도 | 수 |
|-------|----------------|--------|-----|
| royal | 430~460 | 5 | 12 |
| legend | 600~620 | 5 | 3 |
| villain | 380~445 | 3~4 | 6 |
| normal | 280~340 | 2~3 | 20 |

## 스탯 범위 가이드라인

- 개별 스탯 범위: **25 ~ 120**
- 신규 엔트리 추가 시 `class` 기준 총합 목표를 지킬 것
  - normal: 270 ~ 350
  - villain: 370 ~ 460
  - royal: 420 ~ 470
  - legend: **600 이상 필수**
- 스킬 개수: 일반 4개, 레전드 5개 권장
- 빛/어둠 이중 타입은 서사적 의미가 있을 때만 부여 (달님·악동 등)
